# ScanPro
ScanPro est une application de numérisation intelligente . Elle permet à n'importe qui de scanner un document papier, extraire le texte automatiquement, le traduire et l'exporter en PDF ou Word, directement depuis son téléphone ou ordinateur. Pas d'abonnement. Pas de connexion requise pour l'OCR.  Simple, rapide, gratuit.
